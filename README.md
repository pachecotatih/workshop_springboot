# workshop_springboot

API REST desenvolvida com Spring Boot.

## Descrição

Esta aplicação é uma API REST em Spring Boot que expõe endpoints para gerenciamento de recursos.

## Funcionalidades

- Criação, leitura, atualização e exclusão de entidades
- Endpoints RESTful
- Estrutura típica de um projeto Spring Boot

## Tecnologias

- Java
- Spring Boot
- H2
- Spring Web

## Como executar

1. Verifique se o Java está instalado.
2. Execute o projeto com `mvn spring-boot:run` ou a partir da sua IDE.
3. Acesse os endpoints via HTTP.
